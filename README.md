# IT Portfolio

A personal portfolio built with **Next.js** to showcase projects, skills, and technical experience.

---

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS

---

## Getting Started

Install dependencies:

```bash
npm install

Run the development server:

npm run dev
# or
yarn dev
# or
pnpm dev

Open:

http://localhost:3000

Project Structure
app/page.tsx – Main landing page
app/layout.tsx – Global layout
components/ – Reusable UI components
styles/ – Global styles (if applicable)

The app auto-updates as you edit files.

Build for Production
npm run build
npm run start
